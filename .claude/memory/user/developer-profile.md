## Developer Profile

**Role**: Full-stack developer specializing in Next.js, React, and TypeScript
**Project**: St. Elizabeth High School website - Walker School-inspired redesign
**Experience Level**: Senior - comfortable with complex animations, design systems, and modern React patterns

**Current Focus**: 
- Systemic design system enforcement (tokens, components, patterns)
- Accessibility and mobile responsiveness improvements
- Technical debt cleanup and architecture refinement

**Preferences**:
- Prefers component-based architecture with clear separation of concerns
- Values design token consistency and type safety
- Prioritizes accessibility (WCAG AAA) and mobile-first approach
- Avoids scroll-jacking anti-patterns in favor of native scroll with progressive enhancement
- Prefers CSS custom properties over hardcoded values

**Technical Stack**:
- Next.js 14 App Router (static export)
- TypeScript 5.x with strict mode
- Tailwind CSS v4
- GSAP 3.x for animations
- Framer Motion for React animations

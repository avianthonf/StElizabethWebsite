| User motion preference → Animation system | Media query value could change during session; component must adapt without reload |

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|

| T-01-07 | Elevation of privilege (motion override) | prefers-reduced-motion detection | mitigate | read-only matchMedia query; no user-supplied code execution |
| T-01-08 | Information Disclosure (event timing) | Single-fire hasAnimated guard may lock too early if user scrolls quickly in first 12% | accept | Walker spec dictates single-pulse hero animation; early lock is by design to prevent jarring re-triggers |
</threat_model>

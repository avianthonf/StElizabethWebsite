<task type="auto">
  <name>Task 2: Create Breadcrumbs component</name>
  <files>src/components/ui/Breadcrumbs.tsx</files>
  <action>
Create a reusable Breadcrumbs component that displays navigation hierarchy.

Component should:
- Accept items prop: array of { label: string, href: string }
- Render Home link first (always links to /)
- Render separator (/) between items
- Render last item as plain text (current page, not a link)
- Use semantic HTML with <nav> and aria-label="Breadcrumb"
- Style with Tailwind classes matching site design (text-sm, maroon accent color)

Example structure:
```tsx
interface BreadcrumbItem {
  label: string;
  href: string;
}

interface BreadcrumbsProps {
  items: BreadcrumbItem[];
}

export function Breadcrumbs({ items }: BreadcrumbsProps) {
  return (
    <nav aria-label="Breadcrumb" className="...">
      <ol className="flex items-center gap-2 text-sm">
        <li><Link href="/">Home</Link></li>
        {items.map((item, index) => (
          <li key={item.href}>
            <span className="mx-2">/</span>
            {index === items.length - 1 ? (
              <span>{item.label}</span>
            ) : (
              <Link href={item.href}>{item.label}</Link>
            )}
          </li>
        ))}
      </ol>
    </nav>
  );
}
```

Use Link from next/link for navigation. Style links with hover:text-[var(--color-primary-maroon)] to match site theme.

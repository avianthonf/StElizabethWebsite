## Phase Context

Phase 2 created Contact and Admissions pages with disabled form placeholders. This plan activates those forms with:
- All required fields per FORM-01 (name, email, phone, message, inquiry type dropdown)
- Honeypot spam protection per FORM-03 (hidden field, CSS to hide, client-side check)
- Accessible validation per FORM-04 (aria-live regions, focus management, keyboard navigation)

Forms will NOT submit yet (Plan 03-02 wires to Formspree). This plan focuses on form structure, validation, and accessibility.

## Design Tokens

Use existing SOP-001 tokens from `src/app/globals.css`:
- `var(--color-primary-maroon)` for error text and focus states
- `var(--color-text-dark)` for labels
- `var(--color-text-body)` for help text
- `var(--color-bg-light)` for form container background

## Validation Requirements

Per WCAG 2.1 AA (Phase 1 established this standard):
- Required fields marked with `*` in label and `aria-required="true"`
- Error messages in `aria-live="polite"` region (announced to screen readers)
- Focus moves to first error field on validation failure
- Error messages associated with fields via `aria-describedby`
- Keyboard navigation: Tab through fields, Enter to submit, Escape to clear errors

## Honeypot Implementation

Per FORM-03, add hidden field to detect bots:
- Field name: `website` (common bot target)
- Hidden via CSS: `position: absolute; left: -9999px; opacity: 0; pointer-events: none`
- Label: "Leave this field blank" (for accessibility, but hidden from visual users)
- Client-side check: if field is filled, block submission (bot detected)

```typescript
export interface ContactFormData {
  firstName: string;
  lastName: string;
  email: string;
  phone: string;
  subject: string;
  message: string;
  website: string; // honeypot
}

export interface FormErrors {
  [key: string]: string;
}

export function validateContactForm(data: ContactFormData): FormErrors {
  const errors: FormErrors = {};
  
  if (!data.firstName.trim()) errors.firstName = 'First name is required';
  if (!data.lastName.trim()) errors.lastName = 'Last name is required';
  
  if (!data.email.trim()) {
    errors.email = 'Email is required';
  } else if (!/^[^\s@]+@[^\s@]+\.[^\s@]+$/.test(data.email)) {
    errors.email = 'Please enter a valid email address';
  }
  
  if (!data.subject) errors.subject = 'Please select a subject';
  if (!data.message.trim()) errors.message = 'Message is required';
  
  return errors;
}

export function isHoneypotFilled(honeypotValue: string): boolean {
  return honeypotValue.trim().length > 0;
}
```

  const { name, value } = e.target;

  // Clear error for this field when user starts typing

  // Honeypot check
  if (isHoneypotFilled(formData.website)) {
    console.warn('Honeypot triggered - bot detected');
    return; // Silently fail for bots

  // Validate
  const validationErrors = validateContactForm(formData);
  if (Object.keys(validationErrors).length > 0) {
    setErrors(validationErrors);
    // Focus first error field
    setTimeout(() => firstErrorRef.current?.focus(), 100);

};
